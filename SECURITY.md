# Security Policy

## Public Demo Boundary

This repository is for synthetic EDI/X12 validation examples only.

Do not commit:

- PHI or patient records.
- Production claim files.
- Payer files.
- Clearinghouse files.
- API credentials.
- App passwords.
- Private keys.
- Certificates.
- Real payer IDs from private contracts.

## Reporting

If you find a security issue, report it privately to the repository owner. Do not open a public issue containing sensitive data.
